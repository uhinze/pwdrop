Most important security features:

  • Encrypted communication using HTTPS
  • Encrypted storage using AES256 Server-Side Encryption
  • Optional client-side encryption using password (so not even admins can read your password).
  • Configurable deletion time and maximum view count.
  • Optional manual deletion at any time.
  • Best practice HTTP security headers.
  • Minimum attack surface by using as few third-party libraries as possible.
